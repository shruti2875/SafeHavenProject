🛡️ SafeHaven – Domestic Violence Detection & Alert System

An intelligent mobile-based safety system that enables users to send real-time emergency alerts during domestic violence situations. The application integrates secure authentication, instant notifications, and communication services to ensure quick response and user safety.

🚀 Features
Emergency alert trigger from mobile app
Secure user authentication
Real-time SMS and call alerts
Push notifications for rapid response
Encrypted data transmission
Simple and user-friendly interface


🏗️ System Architecture
Mobile app sends alert request
Backend processes and verifies user
Notifications triggered via APIs
Alerts sent through SMS / Call and push notifications
🛠️ Tech Stack

Frontend:
React Native / Flutter

Backend:
Node.js (Express) / Django

Database:
MySQL

Security:
JWT Authentication
SSL Encryption
AES Encryption

APIs / Services:
Firebase (Authentication & Notifications)
Twilio (SMS / Call Alerts)
